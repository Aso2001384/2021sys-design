```
@startuml
[*] -> トップページ
トップページ : do/商品一覧を表示
トップページ -> トップページ : 商品検索
トップページ --> 会員情報 : 会員情報をクリック
トップページ -left-> ログイン : ログインをクリック
トップページ -> 商品詳細 : 商品をクリック
トップページ -> カート内 : カートをクリック
トップページ -left-> 問合せフォーム : 問合せをクリック
問合せフォーム --> トップページ : トップページをクリック

会員情報 --> トップページ : トップページをクリック
会員情報 --> 購入履歴 : 購入履歴をクリック
会員情報 -> 会員情報編集 : 編集するをクリック
会員情報 -> 会員情報 : 購入履歴をクリック
会員情報編集 -left-> ログアウト : ログアウトをクリック
会員情報編集 --> 会員情報 : 戻るをクリック

ログアウト --> トップページ : 完了後自動遷移

ログイン : entry/メールアドレス, パスワードを入力
ログイン : do/ログイン認証
ログイン --> トップページ : トップページをクリック

商品詳細 -> カート内 : カートに入れるをクリック
カート内 -up-> トップページ : トップページをクリック/商品検索
カート内 --> 商品詳細 : 商品をクリック

state カート内 {
[*] --> カート
カート -> 購入 : 購入をクリック
カート -> カート : 数量を変更/商品を削除
購入 --> カート : カートをクリック

state 購入手続き {
[*] --> 決済方法選択
決済方法選択 -> 決済確認 : 内容確認をクリック
決済確認 --> 決済方法選択 : 変更をクリック
決済確認 -> 購入完了
 }
}
@enduml
```
